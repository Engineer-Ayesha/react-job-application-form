# Job Application Form
A responsive and interactive Job Application Form built with React.
## Features
* User-friendly interface
* Form validation
* Dynamic error handling
* File upload functionality
* Success message on submission
* Responsive design
* Controlled components using React state
* Skills selection using checkboxes
* Experience selection using radio buttons
## Technologies Used
* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
## Project Purpose
This project was created to practice React form handling, state management, validation, file uploads, conditional rendering, and user experience best practices.
